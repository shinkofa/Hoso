## Travail réalisé

### ✅ LIVRÉ — Fix qualité stream Twitch (Option A)

Commit `16bc88f` — `fix(stream): force CBR + GOP=2s to resolve Twitch UNSTABLE badge`

Diagnostic : badge **UNSTABLE Twitch** à 3500 kbps @ 1080p Fit causé par 2 paramètres encoder non explicites :

1. **Keyframe interval (GOP)** non configuré → default StreamPack 1s, alors que Twitch attend strictement 2s.
2. **Bitrate mode** non forcé → MediaCodec partait en VBR par défaut, alors que Twitch préfère CBR.

Modification chirurgicale dans `MainActivity.kt:configureAndStart` (constructeur StreamPack `VideoConfig`) :
- `gopDurationInS = 2f`
- `customize = { setInteger(KEY_BITRATE_MODE, BITRATE_MODE_CBR) }` (lambda extension `MediaFormat.(Boolean) -> Unit`)

**Validation device par Jay** : « EXCELLENT. Le stream est de meilleure qualité. C'est absolument indéniable. »

### ❌ REVERTÉ — Mixer audio mic + jeu (Option 2)

Tentative d'implémenter `MixedAudioSource : IAudioSourceInternal` orchestrant `MicrophoneSourceFactory` + `MediaProjectionAudioSourceFactory` pour mix PCM 16-bit signed.

**Échecs cumulés** :
1. Premier build : constructeur `VideoCodecConfig` internal utilisé sans le savoir → switch sur le constructeur public avec `Function1<VideoProfileLevelColor.Builder, Unit>`.
2. Deuxième tentative : `MicrophoneSource` et `MediaProjectionAudioSource` sont `internal` Kotlin (javap montrait public au niveau JVM mais Kotlin enforce `internal`) → switch sur les factories publiques.
3. Crash UI au lancement : `MaterialSwitch` (Material3) NPE sur `SwitchCompat.makeLayout` (textOn/textOff null) avec thème `Theme.MaterialComponents` (Material2) → switch sur `SwitchCompat` standard.
4. Switch invisible visuellement (toggle non rendu) sur ColorOS avec thème actuel.
5. Mix produisant du silence (size=0) : bugs convention ByteBuffer — sub-frames retournés en write-state non flippés, output incorrectement flipé.
6. Régression mode mic seul (silence) + cadrage portrait + stream qui ne démarre pas en mode mixer → diagnostic impossible sans logs exploitables.

**Décision** : revert complet du mixer, garder uniquement le fix CBR/GOP. Replanifier l'option 2 sur session fraîche avec investigation préalable du pipeline interne StreamPack.

**Backup conservé** : `app/src/main/java/com/theermite/hoso/audio-backup-revert/MixedAudioSource.kt` — code de référence pour la prochaine tentative (à supprimer ou remettre au propre).
