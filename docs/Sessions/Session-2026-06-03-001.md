### 1. Mockups SVG → PNG @2x
- Constat de départ : 7 mockups SVG (6 écrans + flow d'install) et un `Release-Body-Template.md` existaient déjà en working tree non commité (datés du jour), fidèles à la marque Ermite.

## Erreurs et corrections

| Erreur | Cause | Correction |
|--------|-------|------------|
| Bash tool ≠ PowerShell | Le tool exécute `/usr/bin/bash` (MINGW), pas PowerShell annoncé | Syntaxe bash + chemins `/d/...` |
| Batch Chrome → 0 fichier (×4 tentatives) | (a) profil par défaut verrouillé par Chrome ouvert (b) `--user-data-dir` en chemin MinGW non compris (c) launcher se détache sans pipe (d) **bug quoting bash `\\$name`** mangeait l'expansion de variable dans le chemin de sortie | LOGS FIRST (capture du chemin réel écrit) → root cause = backslash+variable adjacents → **chemin de sortie en forward-slashes** (Chrome Windows les accepte) |
| Images cassées sur Release publiée | SVG non rendus par le proxy camo GitHub | Export PNG + repointage |
| `hoso-release.apk` ≠ asset réel | Template hardcodait un nom inexistant | Référence `.apk` version-proof |
